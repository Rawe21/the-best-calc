# the-best-calc
why think when you can use "the-best-calc". 


# how it works

you type the first number then the operation and then the second number.

it waits for numbers and the operator.
once it has everything it will:
 1. calculates the result (eval..)
 2. rejects anythings thats not in 0-100 (for example if its a -1 it just wont accept it)
 3. guesses the number like an ai till it finds the right one.


# funny little things
 1. theres 20% that it will stop and say that its tired
 2. it wont divide to 0 because the pshysic engine "failed"
 3. everytime it guesses it will go slower and slower (calculators have feelings too yk?)
 4. if you type 0-0 0+0 or 0*0 it will say console.log(HelloWorld)
